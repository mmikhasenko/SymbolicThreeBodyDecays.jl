SymbolicThreeBodyDecays.jl

Public webpage: mmikhasenko.github.io/SymbolicThreeBodyDecays.jl

GitHub repository: github.com/mmikhasenko/SymbolicThreeBodyDecays.jl

This repository provides a Julia code that extends the functionality of the ThreeBodyDecay.jl project to work with SymPy objects. It allows for symbolic computation in the context of three-body decays using the Dalitz-Plot Decomposition.

Demo with Pluto: The demo folder contains Pluto.jl notebooks that provide interactive demonstrations or examples to help users understand the functionality. To run the notebooks, ensure you have Pluto.jl installed and simply open them using Pluto. The html-exported notebooks can be viewed from the webpage.

Features

  • Symbolic Computation: integration with SymPy to perform symbolic calculations using the implementation of algorithms from the ThreeBodyDecay.jl.

Getting Started

  1. Clone the Repository
git clone https://github.com/mmikhasenko/SymbolicThreeBodyDecays.jl.git

  1. Navigate to the demo folder and open the Pluto notebooks for interactive examples.

  2. For detailed functionality, explore the src directory.
